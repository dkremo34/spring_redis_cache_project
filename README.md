# spring_redis_cache_project -

Cache concept with Redis

**Type of cache-**
  1. In Memory 
  2. Distributed

Annotation-
  @Configuration
  @EnableCaching
  @Cacheable(value="nameofcahce", key="#city")
  @CachePut(value="nameofcahce", key="#city")
  @CacheEvict(value="nameofcahce", key="#city")
