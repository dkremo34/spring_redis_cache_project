Cache concept with Redis

**Type of cache-**

  2. Distributed

Annotation-
  1. @Configuration
  2. @EnableCaching
  3. @Cacheable(value="nameofcahce", key="#city")
  4. @CachePut(value="nameofcahce", key="#city")
  5.  @CacheEvict(value="nameofcahce", key="#city")
